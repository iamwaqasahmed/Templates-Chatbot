# Web — Next.js Frontend

Next.js application for the chatbot platform UI.

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev          # http://localhost:3000

# Lint & format
npm run lint
npm run format

# Type check
npm run typecheck

# Run tests
npm test
```

## Project Structure

```
src/
  app/            # Next.js App Router pages
  components/     # Reusable UI components
  lib/            # Client utilities (API client, auth, etc.)
```
